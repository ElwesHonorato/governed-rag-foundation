# Worker `src` Abstraction Matrix

Scope:
- Matrix is derived from worker implementation code under `domains/worker_*/src`.
- Columns are workers.
- Rows are abstraction levels.
- Cell values are the classes/code in worker `src` performing the role.

| Abstraction Level | `worker_scan` | `worker_parse_document` | `worker_chunk_text` | `worker_embed_chunks` | `worker_index_weaviate` | `worker_manifest` | `worker_metrics` |
|---|---|---|---|---|---|---|---|
| **L0 — Composition / Runtime assembly** | `src/app.py::run()` + `ScanServiceFactory.build()` | `src/app.py::run()` + `ParseServiceFactory.build()` | `src/app.py::run()` + `ChunkTextServiceFactory.build()` | `src/app.py::run()` + `EmbedChunksServiceFactory.build()` | `src/app.py::run()` + `IndexWeaviateServiceFactory.build()` | `src/app.py::run()` + `ManifestServiceFactory.build()` | `src/app.py::run()` + `MetricsServiceFactory.build()` |
